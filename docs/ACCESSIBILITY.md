# Accessibility Notes

- A skip link is provided before the header.
- Navigation uses semantic `header`, `nav`, `main`, and `footer` landmarks.
- Interactive controls use visible focus styles.
- Project filters use a labeled search input and `aria-pressed` tag buttons.
- Decorative circuit imagery is hidden from assistive technology.
- Project images and diagrams include alt text from configuration.
- Color contrast is designed around dark surfaces with bright trace and solder accents.

## Maintenance Checklist

- Add meaningful alt text for every new project image.
- Keep project writeups concise and scannable.
- Avoid link text like "click here"; name the destination or action.
- Verify keyboard access when adding new controls.
